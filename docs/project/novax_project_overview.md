# NOVAX ERP - Project Overview

## Business Context
Novax is a company dedicated to manufacturing and distributing cleaning products (detergents, dishwashing liquids, hand soaps, fabric softeners, industrial cleaners, automotive detergents). They target both wholesale and retail markets.

## Project Goal
To develop a modern web platform that serves three types of users:
1. **Visitors:** View the catalog and prices.
2. **Registered Clients:** Register, view catalog/prices, interact via shopping cart, and manage orders.
3. **Administrators:** Full system control (inventory, products, prices, orders, clients, providers).

## Scope
The application includes a scalable backend API and a responsive frontend web application, designed primarily for performance and cross-device ease of use (mobile, tablet, and desktop). It will be expanded in the future with mobile applications and advanced mapping services.
